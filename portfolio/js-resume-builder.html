JavaScript Resume Builder

The focus of this example isn't so much the presentation as it is what is working behind the scenes.

My online degree's final project was to create a resume builder powered by JavaScript. They provided us with a basic template then taught us how to store data in JS objects, create functions to iterate through that data, then display it on the page with jQuery.

Check out the resume builder in action, then see the JS files powering it.
